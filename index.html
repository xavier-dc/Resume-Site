About Me

Based in New York City, I have experience in front-end, back-end, mobile, and game development. My background in live TV and online broadcast includes a strong command of both hardware and software, and directly informs my current skillset. I'm a musician with a passion for gear and new technology, and I love building tools to make people's lives easier.

Projects

    Singular Designer

    As part of a two man team I made a graphics application in Javascript, HTML, and CSS. Using the Singular Graphics SDK, the application encompassed the end-to-end graphics process for live broadcast. Users could create and recall shows, save presets, parse excel documents, and customize and prepare live graphics for broadcasting in real-time.

    Figma Layout Converter

    I created a layout converter tool using React, Redux, Python, and VB.NET. This tool enables the user to design, convert, and import a CSS layout into vMix as XML data, thereby preserving the original aspect ratio of the source material.

    Apartment Package Manager

    I created a package manager application using React that utilizes Express.js and File-stream to write JSON data. The app logs the time and date, and dynamically updates package drop-offs and pickups.

    Wildlife Pigeon Detection AI

    Using Yolov3, Python, and PyTorch, I made an AI that automatically captures and logs visits from NYC pigeons.

    React Messaging App

    I constructed a messaging application using React and Firebase to connect multiple individuals to a server and send/receive messages and reactions.

    Radiant Pulse Fitbit Watch Face

    Using Fitbit Studio, Javascript, and Open-Meteo's open source weather API, I created a free watch face for Fitbit Versa 3 and Fitbit Sense with animations, calorie tracking, heart-monitoring, geolocation and weather reporting.

    CLARIFAI Facial Recognition

    I built a website that handles registering, point system, and sign-in of users and uses an API to detect faces in images. Created utilizing React, PostgreSQL, Heroku, HTML, CSS, Javascript, and numerous frameworks and libraries for styling and optimization as well as CLARIFAI's API.
