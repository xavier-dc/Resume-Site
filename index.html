Projects

    Singular Designer

    As part of a two man team I made a graphics application in Javascript, HTML, and CSS. Using the Singular.live Graphics SDK, the application encompassed the end-to-end graphics process for live broadcast. Users could create and recall shows, save presets, parse excel documents, and customize and prepare live graphics for broadcasting in real-time.

    Figma Layout Converter

    I created a layout converter tool using React, Redux, and Python. This tool enables the user to design, convert, and import a CSS layout into vMix as XML data, thereby preserving the original aspect ratio of the source material.

    Apartment Package Manager

    I created a package manager application using React that utilizes Express.js and File-stream to write JSON data. The app logs the time and date, and dynamically updates package drop-offs and pickups.

    Wildlife Pigeon Detection AI

    Using Yolov3, Python, and PyTorch, I made a script that automatically captures and logs visits from NYC pigeons.

    React Messaging App

    I constructed a messaging application using React and Firebase to connect multiple individuals to a server and send/receive messages and reactions.

    CLARIFAI Facial Recognition

    I built a website that handles registering, point system, and sign-in of users and uses an API to detect faces in images. Created utilizing React, PostgreSQL, Heroku, HTML, CSS, Javascript, and numerous frameworks and libraries for styling and optimization as well as CLARIFAI's API.
